## quickstart

```
npm i -g pnpm lerna
```

```
pnpm i

pnpm run dev

pnpm run publish -- x.x.x
```

[lerna](https://www.lernajs.cn/) 命令

```
lerna run build
lerna changed
lerna diff
```
